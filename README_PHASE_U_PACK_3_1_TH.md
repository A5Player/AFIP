# AFIP Phase U Pack 3.1 — Compact Dashboard 2

แก้ Dashboard 2 จริงให้การ์ด Intelligence และ Engine มีขนาดเท่ากัน อ่านแบบบรรทัดเดียว และไม่ขยายความสูงตามข้อความยาว

- Grid 4 คอลัมน์บนจอกว้าง
- การ์ดสูงเท่ากัน 360px
- หัวข้อ คำอธิบาย ชื่อข้อมูล และค่าอยู่บรรทัดเดียว
- ข้อความยาวใช้ ellipsis และดูข้อความเต็มได้เมื่อวางเมาส์
- ตารางภายในเลื่อนได้โดยไม่ทำให้การ์ดอื่นยืดตาม
- ไม่แก้ Trading Logic, Profile Policy, Lot, SL, TP หรือ Execution
