# Milestone N Pack 6 — Portfolio Drawdown Protection

Adds a deterministic, research-only portfolio safety gate after exposure coordination. It validates equity floor, maximum drawdown, daily loss, and consecutive loss limits. A blocked result prevents new allocation but does not modify existing positions or transmit orders.

Permanent policies remain XM only, GOLD# only, 1 Unit = 0.01 Lot, independent position lifecycles, Protected Runner accounting, and prohibition of DCA, averaging down, Martingale, Grid, and Recovery Trading.
