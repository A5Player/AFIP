- จัดการ์ด Runtime, MT5, Live financial, Fresh data และ Lot policy ให้อยู่แถวเดียว
- ลดขนาดไอคอน ตัวเลข ระยะห่าง และ progress bar
- ข้อความชื่อการ์ดยาวจะไม่ดัน layout แตก
- ไม่แก้ข้อมูล ตาราง MT5 runtime execution หรือ authority ใด ๆ
