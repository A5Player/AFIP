# Milestone P Pack 10 — Market Behaviour Intelligence Complete

Patch นี้ปิด Milestone P ภายใต้ AFIP Version 1.0 Feature Freeze

ตรวจสอบรายการต่อไปนี้:

- Capability lineage ของ Pack 1–9 ครบถ้วน
- `PBEHAV-` Capability ID ไม่ซ้ำ
- Pack 9 ผ่านการรับรอง Manual Review
- Data Quality และ Future Safety
- Deterministic Runtime
- Market Regime ก่อน Market Behaviour
- ลำดับเวลาการปิด Pack ถูกต้อง
- นโยบาย XM / GOLD# / Base Unit 0.01 Lot
- Execution Lock แบบ Simulation-only ถาวร
