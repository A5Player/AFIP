# Milestone N Pack 6 — Portfolio Drawdown Protection

เพิ่มด่านความปลอดภัยระดับพอร์ตแบบ Deterministic และ Research-only หลัง Portfolio Exposure Coordination เพื่อตรวจ Equity Floor, Maximum Drawdown, ผลขาดทุนรายวัน และจำนวนการขาดทุนต่อเนื่อง หากไม่ผ่าน ระบบจะระงับการจัดสรรใหม่ แต่จะไม่แก้ไขสถานะที่มีอยู่และไม่ส่งคำสั่งซื้อขาย

นโยบายถาวรยังคงเป็น XM เท่านั้น, GOLD# เท่านั้น, 1 Unit = 0.01 Lot, Position Lifecycle แยกอิสระ, นับ Protected Runner ใน Exposure และห้าม DCA, Averaging Down, Martingale, Grid และ Recovery Trading
