# AFIP Milestone T Pack 8
## Runtime Standard Adapter, Historical Backfill Orchestration และ Context-Aware Integration Foundation

แพ็กนี้เชื่อมมาตรฐานวิจัยที่ ACTIVE และเจ้าของอนุมัติแล้ว เข้าสู่คำแนะนำ Runtime แบบมีขอบเขต โดยยังไม่ส่งออเดอร์และไม่สามารถข้าม Risk, Trading Cost, Profile Capacity หรือ Execution Permission Gate ได้

เพิ่มตัวจัดการดึงข้อมูลย้อนหลังแบบ Provider-neutral ตั้งแต่ข้อมูลแรกสุดที่หาได้จนถึงแท่งปิดล่าสุด พร้อมเรียงเวลา ตัดข้อมูลซ้ำ เก็บแหล่งที่มา และบันทึกแบบ Append-only
