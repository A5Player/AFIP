# AFIP Milestone T Pack 3

## เครื่องจำลองข้อมูลย้อนหลังและตัวสร้างชุดข้อมูลวิจัย

Milestone T Pack 3 เพิ่มเครื่องมือวิจัยที่เดินข้อมูลแท่งราคาในอดีตทีละแท่งตามลำดับเวลา จุดตัดสินใจจะเห็นเฉพาะแท่งปัจจุบันและข้อมูลก่อนหน้าเท่านั้น แล้วบันทึกผลลงชุดข้อมูลแบบเพิ่มต่อท้ายภายในเขตกักกันงานวิจัย

## สิ่งที่เพิ่ม

- เครื่องเดินข้อมูลย้อนหลังตามลำดับเวลาอย่างเคร่งครัด
- นาฬิกาจำลองและข้อมูลความคืบหน้าแบบกำหนดผลซ้ำได้
- ตัวสร้างภาพรวมตลาดจากข้อมูลที่มองเห็นแล้วเท่านั้น
- ช่องเชื่อมสำหรับสร้างผู้สมัครงานวิจัย
- การตรวจสอบและปรับรูปแบบผู้สมัครให้เป็นมาตรฐาน
- ชุดข้อมูลภาพรวมแต่ละแท่ง
- ชุดข้อมูลผู้สมัคร
- ชุดข้อมูลการตัดสินใจ
- ชุดข้อมูลลำดับเหตุการณ์จำลอง
- ชุดข้อมูลสรุปการวิจัยแต่ละรอบ
- การจัดเก็บ JSONL แบบเพิ่มต่อท้ายเท่านั้น
- สายโซ่ค่าตรวจสอบและการตรวจพบข้อมูลถูกแก้ย้อนหลัง
- การทำงานต่อจากแท่งล่าสุดที่ประมวลผลแล้ว
- ข้อมูลสำหรับหน้าแสดงผลและขนาดเขตกักกันงานวิจัย
- ข้อมูลทุกชุดเริ่มต้นเป็น `EXPERIMENTAL`
- งานวิจัยยังไม่สามารถใช้กับระบบ Production

## ลำดับการทำงาน

```text
ข้อมูลแท่งราคาในอดีต
        ↓
เดินตามลำดับเวลา
        ↓
เห็นเฉพาะข้อมูลปัจจุบันและอดีต
        ↓
สร้างภาพรวมตลาด
        ↓
สร้างผู้สมัครงานวิจัย
        ↓
บันทึกเหตุผลการตัดสินใจ
        ↓
เพิ่มข้อมูลต่อท้ายในเขต Experimental
```

Pack นี้ยังไม่ค้นหาจุดเข้า จุดออก ระยะตัดขาดทุน เป้าหมายกำไร การถือสถานะ หรือการเพิ่มสถานะแบบพีระมิด หน้าที่ของ Pack นี้คือสร้างฐานข้อมูลและเครื่องเดินกราฟที่ปลอดภัยสำหรับ Pack วิจัยถัดไป

## ขอบเขตความปลอดภัย

Pack นี้:

- ไม่เรียกตรวจสอบหรือส่งคำสั่งซื้อขายไปยัง MT5
- ไม่เปิด แก้ไข หรือปิดสถานะ
- ไม่เปลี่ยนขนาด Lot ปัจจุบัน
- ไม่เปลี่ยนนโยบาย TP หรือ SL
- ไม่เปลี่ยนตรรกะการเทรด Production
- ไม่เลื่อนระดับงานวิจัยเป็นความรู้ Production
- ไม่อนุญาตให้ Production Runtime อ่านชุดข้อมูล Experimental

## ความสมบูรณ์ของชุดข้อมูล

แต่ละบรรทัดของชุดข้อมูลประกอบด้วย:

- ชื่อชุดข้อมูล
- ลำดับรายการ
- ค่าตรวจสอบของรายการก่อนหน้า
- เนื้อหารายการ
- ค่าตรวจสอบของรายการปัจจุบัน

หากมีการแก้ข้อมูลเก่า การตรวจสอบสายโซ่จะไม่ผ่าน ส่วนการทำงานต่อจะอ่านลำดับเหตุการณ์เดิมและเริ่มจากแท่งถัดไปที่ยังไม่ถูกประมวลผล
