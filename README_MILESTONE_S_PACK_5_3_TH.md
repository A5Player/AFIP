# AFIP Gold Pro V1.0 — Milestone S Pack 5.3

## การเชื่อมข้อมูล Runtime เข้าศูนย์วิจัย

แพตช์นี้เชื่อม Ledger การทำงานจริงของบัญชีทดลอง P1-P4 เข้ากับฐาน Trade Case จาก Pack 5.2

เพิ่มความสามารถ:
- นำเข้า Ledger แบบไม่บันทึกซ้ำ
- บันทึกสถานะระหว่างถือออเดอร์
- Floating Profit/Loss, MFE และ MAE
- บริบท SL และ TP
- บันทึกการปิดออเดอร์และคุณภาพ Exit
- Profit Retained และ Profit Given Back
- เชื่อมจุดสังเกต M15/M30/H1/H4/D1

Collector เป็น Research Only ไม่สามารถเปิด แก้ไข หรือปิดออเดอร์ และไม่ส่งผลต่อการตัดสินใจเทรด
