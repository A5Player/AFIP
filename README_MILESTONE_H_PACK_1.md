Patch-only dashboard foundation for AFIP. This pack adds presentation metadata for intelligence and engine categories without changing trading logic.

- Dashboard foundation runtime report
- Dashboard blueprint documentation

## Safety

- No live execution
- No trading logic changes
- No risk-rule changes
- No order placement
- Deterministic presentation output only
