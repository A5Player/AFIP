# AFIP Score Dictionary Guide

The canonical registry is `config/research_data/score_dictionary.json`.

Every score exposes its range, raw components, formula identifier, hard gates, and interpretation. Composite confidence is evidence strength—not execution permission. Data freshness, trading cost, risk, margin, profile capacity, execution mode, duplicate protection, and broker validation remain independent gates.

Historical score values retain the formula version used at calculation time. A new weighting or penalty model must receive a new formula version.
