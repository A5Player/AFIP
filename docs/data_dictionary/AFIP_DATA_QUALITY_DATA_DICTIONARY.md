# Data Dictionary

| Field | Meaning |
|---|---|
| assessment_id | Deterministic assessment identifier |
| dataset_id | Stable central catalog dataset ID |
| overall_score | Weighted 0-100 score |
| readiness_level | RESEARCH_READY, CONDITIONALLY_READY, REVIEW_REQUIRED, or NOT_READY |
| usage_restriction | Approved level of research use |
| completeness | Coverage of required records and fields |
| consistency | Internal agreement across records |
| validity | Conformance to schema and rules |
| freshness | Timeliness for intended use |
| lineage | Traceability to data origin and transformations |
| documentation | Guide, schema, and dictionary coverage |
| integrity | Checksum and corruption confidence |
| reasons | Machine-readable limitations |
| execution_authority | Always NONE |
