# Milestone I Pack 1 — Economic Calendar Intelligence

Transforms supplied economic-calendar events into deterministic structured intelligence for XM GOLD#.

## Scope
- Impact: LOW / MEDIUM / HIGH
- Gold relevance and event category
- Countdown and active event window
- Explainable trading calendar block
- Dashboard EN/TH
- Read-only; news never places orders directly

Providers may be connected later. This pack intentionally uses supplied records for deterministic replay, testing, and auditability. Live execution remains disabled.
