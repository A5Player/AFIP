# Milestone I Pack 8 — USD Index Intelligence

## Purpose
Convert supplied USD Index (DXY) observations into deterministic, structured GOLD# market intelligence.

## Scope
- DXY current and previous values
- Percentage change and trend
- Gold correlation classification
- Same-direction divergence detection
- Aggregate USD trend and GOLD# effect
- English/Thai Dashboard explainability

## Safety
This module is context only. It cannot place orders and always returns `execution_allowed=False`. XM only, GOLD# only, live execution disabled.

## Validation
Run `RUN_MILESTONE_I_PACK_8.ps1` or `.bat`.
