# Production Bring-up Pack 7 — Historical Data Manager

Adds a read-only production dashboard view for XM GOLD# historical coverage. It reports M1, M5, M15, H1, H4 and D1 bar counts, missing and duplicate bars, quality score, research readiness, walk-forward readiness and the expected next action. It does not download data, submit orders, or enable live execution.

## Safety
- XM only
- GOLD# only
- Live execution disabled
- Deterministic evaluation
- Existing Historical Data Manager preserved
