# Production Bring-up Pack 7 — ตัวจัดการข้อมูลย้อนหลัง

เพิ่มมุมมองข้อมูลย้อนหลังแบบอ่านอย่างเดียวบน Dashboard สำหรับ XM และ GOLD# แสดงจำนวนแท่ง M1, M5, M15, H1, H4 และ D1 รวมถึงข้อมูลขาดหาย ข้อมูลซ้ำ คะแนนคุณภาพ ความพร้อมด้าน Research ความพร้อม Walk-Forward และขั้นตอนถัดไป ระบบไม่ดาวน์โหลดข้อมูล ไม่ส่งคำสั่ง และไม่เปิด Live Execution

## ความปลอดภัย
- รองรับ XM เท่านั้น
- รองรับ GOLD# เท่านั้น
- ปิด Live Execution
- ผลลัพธ์เป็น Deterministic
- รักษา Historical Data Manager เดิมไว้
