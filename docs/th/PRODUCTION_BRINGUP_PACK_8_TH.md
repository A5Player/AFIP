# Production Bring-up Pack 8 — Dashboard Live Runtime

เพิ่มชั้นตรวจสอบความพร้อมของ Dashboard แบบสดที่ทำงานแบบกำหนดผลได้และอ่านอย่างเดียว สำหรับ XM และ GOLD#

## ขอบเขต
- รอบเวลารีเฟรชและสถานะ Runtime
- ลำดับ Snapshot และความสดของข้อมูล
- ความพร้อมของส่วนประกอบ
- เหตุผลที่รอและการกระทำถัดไปทั้งภาษาอังกฤษและภาษาไทย
- เชื่อม Panel เข้ากับ Dashboard

ยังคงปิด Live Execution และไม่มีการเปลี่ยนตรรกะส่งคำสั่งหรือการตัดสินใจเทรด
