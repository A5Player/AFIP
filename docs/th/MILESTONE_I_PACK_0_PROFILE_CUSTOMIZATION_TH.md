# Milestone I Pack 0 — ปิดงานการปรับแต่งโปรไฟล์

Pack นี้ปิดช่องว่างของ Profile Manager ก่อนเริ่มแพลตฟอร์ม Market Intelligence

## ความสามารถ
- ตั้งชื่อโปรไฟล์และรหัสโปรไฟล์เองได้
- คัดลอกจาก Conservative, Balanced, Growth หรือ Research
- แก้ไขระดับความเสี่ยง จำนวน Unit เงินทุนต่อ Unit การแยกคำสั่ง การถือข้ามคืน และ Research Mode
- บันทึก เปิดใช้งาน คัดลอก เก็บถาวร ลบ และกำหนดบัญชี
- บันทึก JSON แบบ atomic พร้อมประวัติเวอร์ชัน
- Profile ยังคงแยกจาก Account และ MT5
- Dashboard อธิบายเป็นภาษาอังกฤษและภาษาไทย
- Live Execution ยังปิดอยู่

ตำแหน่งจัดเก็บเริ่มต้นคือ `runtime/profiles/profiles.json`
