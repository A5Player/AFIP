# Milestone I Pack 0 — Profile Customization Completion

Completes reusable profile management before Market Intelligence Pack 1.

## Features
- Custom profile names and deterministic IDs
- Copy from Conservative, Balanced, Growth, or Research presets
- Edit risk, Unit count, capital per Unit, split orders, overnight holding, and research mode
- Save, activate, duplicate, archive, delete, and assign an account
- Atomic JSON persistence and version history
- Profile remains separate from account and MT5
- Dashboard explainability in English and Thai
- Live execution remains disabled

Default storage: `runtime/profiles/profiles.json`. A different path may be injected for tests or deployment.
