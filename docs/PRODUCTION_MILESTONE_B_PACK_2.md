# Production Milestone B Pack 2 — Adaptive Weight Engine

Pack 2 introduces adaptive financial intelligence weighting for Milestone B.

## Scope

- AdaptiveWeightEngine
- RegimeWeightProfile
- PerformanceWeightAdjustment
- IntelligenceWeightMatrix
- ProductionMilestoneBWeightRuntime

## Production Requirements

- Additive implementation only
- Backward compatible with existing AFIP modules
- International financial terminology
- Pytest coverage included
- Local quality and GitHub CI compatible

## Runtime Purpose

The runtime converts intelligence contribution, market regime, and performance attribution into normalized allocation weights. These weights can be consumed by fusion and decision runtimes without changing legacy execution behavior.
