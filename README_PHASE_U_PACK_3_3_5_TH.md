# AFIP Phase U Pack 3.3.5

## Dashboard Coverage, Progress, Freshness & Research Status Integration

แพตช์นี้เพิ่มตารางสถานะ Universal Timeframe แบบมีโครงสร้างและ deterministic ลงใน Dashboard 3

แสดงข้อมูลครบสำหรับ M1, M5, M15, M30, H1, H4 และ D1 ได้แก่

- จำนวนแท่งที่มีและแท่งที่ผ่านการตรวจ
- จำนวน Gap และแท่งที่ขาด
- สถานะ Freshness
- จำนวนแท่ง Replay ที่ประมวลผลในรอบล่าสุด
- จำนวนแท่งที่ครอบคลุมและสถานะ Replay Completion
- สถานะ Integrity
- Research Eligibility

Automatic Research Runtime panel จะแสดงหลักฐานสำคัญแบบ scalar แทนการนำ nested dictionary มาแสดงเป็นข้อความยาว ทั้งยังรักษา Dashboard สามหน้าและไฟล์ redirect เดิมไว้เพื่อ Backward Compatibility

แพตช์นี้เป็น Presentation/Research Evidence เท่านั้น ไม่มีอำนาจเปลี่ยนนโยบาย Live Trading และไม่มีการเรียกส่งคำสั่งซื้อขาย
