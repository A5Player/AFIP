# Milestone M Pack 9 — Pattern Confidence

Adds deterministic, bounded research confidence scoring for validated pattern and cluster knowledge.

## Scope
- Confidence score bounded to 0–100
- HIGH / MEDIUM / LOW confidence tiers
- Evidence inputs: sample count, expectancy, profit factor, dispersion, validation status
- Market-regime context and source-lineage validation
- Deterministic IDs and stable ordering
- Research-only authority
- Permanent execution lock

Production knowledge approval remains disabled. No broker request or order transmission is created.
