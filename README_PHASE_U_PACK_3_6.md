# AFIP Phase U Pack 3.6 — Dashboard Authority Consolidation

Introduces one `DashboardAuthority` to coordinate the complete dashboard build. Existing launcher APIs remain backward-compatible adapters.

UI typography is compacted across the command center and split dashboards: smaller icons, balanced headings, single-line truncation with ellipsis, and tighter metric/table sizing.

This pack is presentation-only. It does not start execution or research and does not change trading authorities.
