# AFIP Milestone P Pack 5 — การตรวจเสถียรภาพพฤติกรรมตลาด

แพตช์นี้ตรวจรายงานสถิติการเปลี่ยนผ่านจาก Pack 4 ที่ผ่านการรับรองข้ามหลายช่วงเวลาวิจัยตามลำดับเวลา

ระบบวัดความแปรปรวนของ persistence, regime change, behaviour change, ความสอดคล้องของสถานะหลัก, stable-window rate และความครอบคลุมของ transition พร้อมระงับ lineage ผิด ข้อมูลซ้ำ ความครอบคลุมไม่พอ ลำดับเวลาผิด future leakage ข้อมูลไม่ผ่าน ค่าผิด ความแปรปรวนสูง ความสอดคล้องต่ำ และการละเมิดนโยบายล็อก

คอมโพเนนต์เป็น immutable, deterministic และ research-only ไม่มีสิทธิ์ปรับพารามิเตอร์ เปลี่ยน Trading Logic ส่งเสริม Production Knowledge แก้ Position ติดต่อ Broker หรือส่ง Order
