# Phase U Pack 3 — Three Dashboards and Real Data Integrity

Creates three separate HTML dashboards:

1. `afip_profiles_dashboard.html` — P1-P4 operations and all account/financial data, auto refresh every 5 seconds.
2. `afip_intelligence_engine_dashboard.html` — Intelligence and engines, manual refresh.
3. `afip_research_data_dashboard.html` — Research inventory, dataset evidence, real Top 10 and expandable Top 100, manual refresh.

Financial placeholder values were removed from the default dashboard record. Missing financial values are displayed as `DATA_UNAVAILABLE`.

This patch is presentation and evidence only. It does not change trading logic or execution authority.
