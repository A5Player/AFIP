# Milestone M Pack 3 — Pattern Similarity Search

เพิ่มระบบค้นหา Pattern ที่คล้ายกันแบบ Deterministic และแยกตาม Market Regime บน Pattern สำหรับการวิจัยที่ผ่าน Canonical Schema พร้อม Cosine Similarity ที่ควบคุมขอบเขต การจัดอันดับคงที่ การตรวจ Lineage และการป้องกัน Future Leakage

Pack นี้ใช้เพื่อ Research เท่านั้น ไม่อนุมัติ Production Knowledge ไม่สร้าง Broker Request และไม่ส่งคำสั่งซื้อขาย
