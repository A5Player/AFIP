# AFIP Milestone T Pack 14

## ความต่อเนื่องแบบไม่ต้องเฝ้า การตรวจสอบหลังเริ่มระบบใหม่ และการควบคุมการกู้คืน

Pack 14 เพิ่มขอบเขตตรวจสอบความต่อเนื่องเมื่ออินเทอร์เน็ตขัดข้อง การเชื่อมต่อ MT5 ขัดข้อง โปรแกรมเริ่มระบบใหม่ หรือ VPS เริ่มระบบใหม่ ระบบต้องเปรียบเทียบสถานะบัญชีจริงกับบัญชีรายการแบบเพิ่มต่อท้ายก่อนดูแลสถานะเดิมหรือรับความเสี่ยงใหม่

หลักสำคัญ:

- ห้ามคาดเดาสถานะออเดอร์หลังการขัดข้อง
- การเริ่มระบบใหม่ต้องตรวจสอบบัญชีและสถานะที่เปิดอยู่
- สถานะที่ไม่รู้จักหรือขาดหายต้องระงับการดำเนินงาน
- การพบสถานะที่เปิดด้วยตนเองต้องเข้าสู่สถานะปลอดภัย
- หลักฐานบัญชีรายการไม่สมบูรณ์ต้องให้ผู้ดูแลตรวจสอบ
- สิทธิ์ส่งคำสั่งซื้อขายยังคงเป็นเท็จ

Pack นี้ไม่ส่ง ไม่แก้ไข และไม่ปิดคำสั่งซื้อขาย MT5
