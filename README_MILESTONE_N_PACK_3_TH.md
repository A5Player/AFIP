# Milestone N Pack 3 — Portfolio Risk Engine

เพิ่มระบบประเมินความเสี่ยง Portfolio แบบ Deterministic และ Research-only โดยรวมความเสี่ยงของ Position ที่เปิดอยู่กับความเสี่ยงที่เสนอ พร้อมตรวจ Risk Budget, Drawdown, Margin Level, จำนวน Unit รวม, Position Lineage, วงจรชีวิต Position แยกอิสระ, Exposure ของ Protected Runner และนโยบายวิธีต้องห้ามถาวร

Execution ยังคง `LOCKED_SIMULATION_ONLY` และไม่มีการสร้างคำขอ Broker หรือส่ง Order
