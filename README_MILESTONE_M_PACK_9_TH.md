# Milestone M Pack 9 — ความเชื่อมั่นของ Pattern

เพิ่มระบบคำนวณค่าความเชื่อมั่นสำหรับ Pattern และ Cluster แบบ Deterministic และมีขอบเขต 0–100

## ขอบเขต
- ระดับ HIGH / MEDIUM / LOW
- ใช้จำนวนตัวอย่าง, Expectancy, Profit Factor, Dispersion และสถานะ Validation
- ตรวจ Market Regime และ Source Lineage
- รหัสและลำดับผลลัพธ์คงที่
- ใช้สำหรับ Research เท่านั้น
- คง Execution Lock ถาวร
