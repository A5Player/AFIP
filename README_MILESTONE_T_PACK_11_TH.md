# AFIP Milestone T Pack 11

## Complete Trade Plan, Capital Stewardship และ Unattended Safety Foundation

Pack นี้สร้างอำนาจรับรองแผนการเทรดแบบครบวงจร โดยยังไม่แตะการส่งออเดอร์จริง

## กฎถาวร

`NO_COMPLETE_PLAN_NO_ORDER`

ก่อนพิจารณาเปิดออเดอร์ ต้องมีแผนครบ 6 ส่วน:

1. แผนวิเคราะห์สถานการณ์ตลาด
2. แผนเข้าออเดอร์
3. แผนบริหารเงินทุน
4. แผนดูแลตำแหน่ง
5. แผนออกและรักษากำไร
6. แผนรับมือความผิดพลาดและกู้คืนระบบ

จำนวนไม้ที่อนุญาตใช้ค่าต่ำสุดจาก Capital, Risk, Margin, Exposure, Correlation และ Profile Capacity ความมั่นใจของสัญญาณไม่มีอำนาจกำหนดจำนวนไม้

ระบบเก็บ Complete Plan, Certification และ Lifecycle Event แบบ Append-only พร้อม Data Contract สำหรับ Dashboard สองหน้า ไม่มี MT5 Order Sender และไม่ข้าม Safety Gate ใด ๆ
